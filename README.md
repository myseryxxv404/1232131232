## Описание программы
Random Task Generator (Генератор случайных задач) - это графическое приложение, разработанное на Python с использованием библиотеки Tkinter. Оно предназначено для генерации случайных задач из предопределенного списка. Приложение позволяет добавлять новые задачи, фильтровать их по типу, отображать сгенерированную задачу и вести историю выполненных задач. Данные (список задач и история) сохраняются в формате JSON для возможности загрузки при последующих запусках.
---
## Требования для запуска
Что нужно установить на компьютер:
- Python 3.10 или выше
- Библиотеки: `tkinter', 'random', 'json','os'.
